Programa desenvolvido em python 3.10, em ambiente windows

Utiliza bibliotecas numpy, sklearn e pandas

Para executar um código basta ir para a pasta equivalente e executar o comando "python nome_do_programa.py" ou "py nome_do_programa.py"
Obs: usar python3, código não foi testado em python2

Os resultados de todos os programas, com a exceção do primeiro programa (./Codigos/1/Analise_estatica.py), usam de algóritmos aleatórios, 
portanto resultados não serão os mesmos dos descritos na documentação, porém devem ser parecidos. O exercício 5 foi resolvido com o mesmo 
código do exercício 3, portanto não há uma pasta 5 na pasta Codigos.
